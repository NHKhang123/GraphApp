# CT239E - Niên luận cơ sở ngành KTPM

Ứng dụng vẽ đồ thị - Thuật toán tìm cây khung tối tiểu

## Công nghệ

- **Backend:** Java
- **Frontend:** JavaFX

## Cài đặt


### Thư mục gốc

- Tải file .zip từ repo dự án và giải nén vào thư mục mong muốn.
- Hoặc clone dự án về từ repo dự án vào thư mục mong muốn.

## Sử dụng

- Chạy file .exe (hoặc file .bat nếu file .exe không khởi chạy được).
- **Lưu ý: Nếu chạy file .bat thì sau khi thoát ứng dụng nên tắt màn hình cmd (terminal) được mở khi bắt đầu chạy ứng dụng.**
